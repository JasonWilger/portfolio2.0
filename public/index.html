My Motivation

Construction worker turned Software Engineer

My most previous career was a sheet metal worker for Georgia's Sheet Metal Local 85. I traveled all over Georgia working on new and old office buildings, renovating hospitals for COVID-19 patients, and much more with the goal of providing a solution to air inefficiency and quality. Throughout this career I realized that along with solving problems and contributing to new projects, I mostly loved providing complex solutions to complex problems. When I was introduced to code just a little under a year ago I started making connections between construction and code. I saw that they were both very similar, but with code I was able to push my self further creatively to provide solutions to a larger scale of problems. This sparked my journey to learning code and lead me to enroll in DigitalCrafts coding bootcamp where I learned a multitude of languages, libraries, and frameworks to help me launch a career in web development.

    Projects

    re-Fresh

    re-Fresh is an online thrift store that was created as a solo frontend project for DigitalCrafts. This web application is built on React and utilizes React Redux to manage state. One of the main features is a shopping cart that holds items added from the product page and totals the items prices together. There is also a checkout feature that simulates a real shipping and payment process. All of the elements are structured and styled with React-Bootstrap making the application clean and responsive.

    Crushn' is a dating app that was created as a team project from DigitalCrafts. This was a backend project with responsibilities that included creating a server with Express, setting up a database with PostgreSQL, and learning how to use Beekeeper Studio and Postman.
